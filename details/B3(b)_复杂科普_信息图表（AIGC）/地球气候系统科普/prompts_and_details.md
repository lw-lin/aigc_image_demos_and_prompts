# prompt

制作一张超级复杂、信息密度极高、采用漫画呈现方式的科普知识介绍海报。整体风格结合日式教育漫画 + 欧美 explainer comic，画面分割成多个独立分镜，每个分镜都有角色讲解、场景示意图、注释框、流程箭头与大量中文科普知识点。

【整体结构】
- 至少 12～20 个分镜
- 多条叙事线并行展开
- 分镜之间用箭头、光线、数据流连接
- 各区块承载不同知识模块
- 有主角（科学家/小助手/可爱形象）逐一解说
- 每个分镜包含背景插图 + 说明文字 + 漫画对白

【视觉风格】
- 日式科普漫画（如《工作细胞》《理科系漫画》）与
  欧美 explain-comic（如《Kurzgesagt》《Randall Munroe》）混合风格
- 色彩丰富但不混乱：蓝 / 黄 / 粉 / 青 / 紫 / 红搭配
- 清晰分栏、边框、圆角对白框
- 大量图标、手绘箭头、示意图、流动路径
- 视觉冲击强，但阅读路线清晰

【内容表现（自动适配主题）】
- 角色以夸张表情解释知识点
- 透视图、爆炸图、断层结构、微观视角
- 流程图：起因 → 过程 → 结果 → 延伸
- 数据框：关键数字、概念解释
- 误区纠正区：对话气泡内“常见误解 vs 真相”
- 微观层面与宏观层面交替呈现
- 交互式箭头串联整页逻辑

【科普信息风格】
- 全中文
- 简化复杂概念、类比解释
- 每个分镜 1–2 个知识要点
- 包含：定义 / 原理 / 应用 / 案例 / 延伸知识

【整体要求】
- 超级复杂 + 超级丰富 + 超级密度
- 但必须结构清晰、能看懂、不乱
- 像专业科普机构 + 教育漫画家共同创作的风格
- 适合做小红书爆款科普图、PPT 扉页、公众号头图
